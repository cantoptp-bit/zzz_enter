# PEGASUS Enter

Static enter/landing page. Deploy to Vercel.

## Vercel: background video (Git LFS)

The background uses an MP4 file stored with **Git LFS**. For the video to appear on Vercel:

1. Open your project on [Vercel](https://vercel.com) → **Settings** → **Git**
2. Turn **on** **Git LFS**
3. **Redeploy** the project (Deployments → ⋮ → Redeploy)

If LFS is off, only the small pointer file is deployed and the video will not load (the fallback image will show instead).
